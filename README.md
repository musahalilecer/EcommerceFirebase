# E-Commerce App - Android (Kotlin, Jetpack Compose & Firebase)

A feature-rich, scalable e-commerce mobile application built with **Kotlin** and **Jetpack Compose**, leveraging **Firebase** for backend services. This project demonstrates the integration of real-time data synchronization, secure user authentication, and modern UI components to provide a complete shopping experience.

## 🚀 Key Features

* **Real-time Product Catalog:** Dynamic product listing and updates powered by Firebase Firestore.
* **User Authentication:** Secure sign-up and login functionality using Firebase Auth.
* **Shopping Cart System:** Persistent cart management allowing users to add, remove, and update products.
* **Order Management:** Seamless checkout flow and order history tracking.
* **Category Filtering:** Efficient product discovery through categorized browsing.
* **Cloud-Synced Data:** User profiles, favorites, and cart data are synced across devices in real-time.
* **Modern Material Design:** Fully responsive UI built with Jetpack Compose and Material 3 components.

## 🏁 Getting Started

### Prerequisites
* Android Studio Ladybug or newer
* JDK 17+
* A Firebase Project (with `google-services.json` configured)

### Installation
1.  Clone the repository:
    ```bash
    git clone [https://github.com/musahalilecer/EcommerceFirebase.git](https://github.com/musahalilecer/EcommerceFirebase.git)
    ```
2.  Add your `google-services.json` file to the `app/` directory.
3.  Open the project in **Android Studio**.
4.  Sync Gradle and run the app on an Emulator or Physical Device.
